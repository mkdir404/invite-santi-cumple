# Minion Invite Landing (React + Tailwind + Vite)

Landing page basada en el PDF de invitación (2 páginas) y lista para desplegar en Vercel.

## Requisitos
- Node.js 18+ (recomendado)

## Correr en local
```bash
npm install
npm run dev
```

## Build / Preview
```bash
npm run build
npm run preview
```

## Deploy en Vercel
- Framework: **Vite**
- Build Command: `npm run build`
- Output Directory: `dist`
